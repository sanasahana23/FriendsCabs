## ğŸŒ Deployment to GitHub

To deploy this project on GitHub Pages:

1. **Create a new repository** on GitHub.
2. **Push your code** to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/sanasahana23/FriendsCabs.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to your repository **Settings**.
   - Navigate to **Pages** in the left sidebar.
   - Under **Build and deployment**, select the `main` branch as the source.
   - Click **Save**. Your site will be live at `https://sanasahana23.github.io/FriendsCabs/`.
